# 시작하기

## 설치법
command-line 도구에서 가장 최근 안정 버전을 설치하려면:

	sudo npm -g install sails

윈도우에서는, sudo가 필요 없습니다:

	npm -g install sails

## 새로운 Sails 프로젝트 만들기 
새로운 어플리케이션 만들기:

	sails new testProject

서버를 시작하는 방법:

	cd testProject
	sails lift

여기에서는, (http://localhost:1337/)로 접속하면, 기본 홈페이지를 볼 수 있을것입니다.

이제, 멋진 작업을 위해 Sails를 받아보세요.
